Metservice

What is it

This is app collates photos that people take of the weather around them and lines them up using the location data to create a collage of images that will look like a panorama of the city.

Who is it for

The app will be avalible in any mobile and table device so it can be used by anyone anywhere so that there is a wide range of photos to collate and give a better picture of the weather in that locations. As for the people who will use the app, it is designed so that anyone can use it because it is simple to do.

How it is used

When someone takes a photo using the app it is uploaded to the database with the location data and then is lined up depending on the location to make a panorama image. The data will be GPS coordinates because it will hopefully easier to map out where the photos will fit into the collage to get the best picture and the mosr accurate weather readings. This is called geotagging which you can find more about here: https://en.wikipedia.org/wiki/Geotagging

Why use it

This app lets you see the weather of any place in realtime with accurate readings but in a way that is more visually pleasing than icons. These images can be injoyed by anyone and shared not just to know the weather but to see the beauty of the city. Potentially these images could be used by weather programs and news channels to give more context to the weather that they explaining.
